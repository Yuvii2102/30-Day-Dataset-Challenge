                                              Day 06 – Social Media User Activity Dataset
                                       
                                       
* Dataset Source :  https://www.kaggle.com/datasets/sadiajavedd/social-media-user-activity-dataset

* Dataset Description
  
  This dataset contains information about social media users such as :
  
  - Age
  
  - Gender
  
  - Country
  
  - Lifestyle
  
  - Screen time
  
  - Likes
   
  - Followers
  
  - Engagement details.
  

* Type of Data
   Structured Data (Tabular Data)
   (Data is stored in rows and columns)
  
* ML Concept
  Machine Learning – analyzing patterns and predicting user engagement and behavior.
  
* Problem Statement
  Can we analyze and predict user behavior based on social media usage, lifestyle, and demographic features?

* Input

  - Age
  
  - Gender

  - Country
  
  - Screen Time
  
  - Likes
  
  - Followers
  
  - Income Level
  
  - Education Level
  
  - Employment Status

  
* Output

  - User Engagement Level
  
  - User Behavior Prediction
  
  
* What We Learned
  
  - Younger users spend more time on social media
  
  - Higher screen time is linked to higher engagement
  
  - Likes and followers strongly affect user popularity
  
  - Lifestyle and income level influence social media usage
  
  - Demographics play an important role in user behavior

  
* Final Outcome
  
  - By analyzing this dataset, we understood how different user factors affect social media engagement. The dataset helps in predicting user behavior using data patterns.


* Real-Life Use
  
  - Used by social media companies to improve recommendations
  
  - Helps marketers target the right audience
  
  - Used to analyze user addiction and engagement
  
  - Helps platforms improve content strategy
